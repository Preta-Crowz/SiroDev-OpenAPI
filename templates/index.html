post로 전송할 내용이 없는 경우에만 get 요청을 사용 가능합니다. 그 외의 경우는 post 요청만을 허용합니다.
오류 신고 및 문의 : https://open.kakao.com/me/nko

스퀘어크립트 API

  • /sqc/encode - 입력받은 문자열을 암호화합니다.
    • query - 암호화할 문자열입니다.
  • /sqc/decode- 입력받은 암호를 복호화합니다.
    • query - 복호화할 문자열입니다.

기타 챗봇 유틸리티

  • /ratio - 두 문자열을 받아, 유사도를 검사합니다.
    • base - 비교할 첫 번째 문자열입니다.
    • compare - 비교할 두 번째 문자열입니다.
  • /lang - eval에서 사용 가능한 언어 목록을 확인합니다.
  • /eval - 주어진 코드를 실행합니다. Paiza API를 이용합니다.
    • code - 실행할 코드입니다.
    • lang - 해당 코드를 실행할 언어입니다.
    • stdin - 입력이 필요한 코드에서 입력할 내용입니다. (필수 아님)
