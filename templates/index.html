SiroDev OpenAPI

post로 전송할 내용이 없는 경우에만 get 요청을 사용 가능합니다. 그 외의 경우는 post 요청만을 허용합니다.
오류 신고 및 문의 : https://open.kakao.com/me/nko

끄투 쿼리 API

  • /urls- 등록된 모든 서버의 정보를 불러옵니다. 삭제를 원하시면 담당자에게 연락해주세요.
  • /word- 해당 단어의 뜻을 불러옵니다.
    • word - 뜻을 불러올 단어입니다.
    • listing - 리스트 반환 여부입니다. 1 혹은 0만 허용되며, 기본값은 0입니다.
  • /status- 등록된 모든 서버의 상태를 불러옵니다.

프로젝트 끄투 API 프록시

  • /starts - 특정 문자열로 시작하는 단어를 모두 불러옵니다.
    • query - 검색 기준이 되는 문자열입니다.
  • /ends - 특정 문자열로 끝나는 단어를 모두 불러옵니다.
    • query - 검색 기준이 되는 문자열입니다.

스퀘어크립트 API

  • /sqc/encode - 입력받은 문자열을 암호화합니다.
    • query - 암호화할 문자열입니다.
  • /sqc/decode- 입력받은 암호를 복호화합니다.
    • query - 복호화할 문자열입니다.

기타 챗봇 유틸리티

  • /ratio - 두 문자열을 받아, 유사도를 검사합니다.
    • base - 비교할 첫 번째 문자열입니다.
    • compare - 비교할 두 번째 문자열입니다.
  • /eval - 주어진 코드를 실행합니다.
    • code - 실행할 코드입니다.
    • lang - 해당 코드를 실행할 언어입니다.
      • /lang - 사용 가능한 언어 목록을 확인합니다.
    • stdin - 입력이 필요한 코드에서 입력할 내용입니다. (필수 아님)
